Student Sphere

B.Tech Final Year Group Project · 2025–26

Student Sphere is a full-stack academic management system built as a B.Tech final year project. It connects students, faculty, and administrators — enabling real-time management of attendance, marks, assignments, notes, and lab reports, backed by a live MongoDB cloud database with JWT authentication and deployed on Netlify & Render.

🚀 Project Highlights

✓ Role-based auth — Student, Faculty & Admin
✓ Real MongoDB Atlas cloud database
✓ JWT secured REST API — Node.js + Express
✓ Bulk attendance & marks management
✓ File uploads — notes, assignments, lab reports
✓ Dark & Light theme with smooth toggle
✓ Fully responsive — mobile to desktop
✓ Live deployed on Netlify + Render

🏗️ Tech Architecture

✓ Frontend — Pure HTML5, CSS3, Vanilla JS · Hosted on Netlify (CDN)
✓ Backend — Node.js + Express 5 REST API · Hosted on Render
✓ Database — MongoDB Atlas cloud · Mongoose ODM
✓ Security — bcrypt password hashing · JWT tokens (7-day expiry)
✓ Access Control — Admin · Faculty · Student role-based permissions
