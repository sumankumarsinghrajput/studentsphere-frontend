Student Sphere

B.Tech Final Year Group Project  ·  2025–26

A full-stack academic management system connecting students, faculty, and administrators. Real-time attendance, marks, assignments, notes & lab reports — backed by MongoDB Atlas, secured with JWT, and deployed live on Netlify & Render.

🚀 Project Highlights

✓ Role-based auth — Student, Faculty & Admin
✓ Real MongoDB Atlas cloud database
✓ JWT secured REST API on Node.js + Express
✓ Bulk attendance & marks management
✓ File uploads — notes, assignments, lab reports
✓ Semester-wise student promotion with undo
✓ Dark & Light theme toggle
✓ Fully responsive — mobile to 4K
✓ Live deployed — Netlify + Render
✓ Hidden easter egg 🥚

🏗️ Tech Architecture

🌐
Frontend
Client Side
HTML5, CSS3, Vanilla JavaScript
No frameworks used. Custom responsive design system with dark/light themes. Deployed on Netlify CDN.
⚙️
Backend
Server Side
Node.js + Express 5 REST API
JWT authentication, bcrypt password hashing, role-based access control. Deployed on Render.
🗄️
Database
Cloud DB
MongoDB Atlas + Mongoose ODM
Cloud-hosted NoSQL database. Direct connection string for reliable ISP compatibility. Stores users, attendance, marks, files.
